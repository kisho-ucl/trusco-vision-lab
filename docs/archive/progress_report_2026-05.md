# Progress Report 2026-05

## 報告の主旨

現在は、映像ベースの作業認識を修論にどう組み込むかを検討するため、関連研究調査を進めている。

特に、俯瞰固定カメラ映像から複数人物を追跡し、各人物IDに対して作業ラベルを推定する問題について、既存技術でどこまで実現できており、どこに研究課題が残るかを整理している。

この調査を先に行っている理由は、以前の論文投稿・査読対応で、手法の新規性や既存研究との差分を十分に説明する必要性を感じたためである。
特に映像ベースの行動認識は、人物検出、追跡、ReID、行動認識、時空間行動検出など既存研究が非常に多いため、単に「倉庫映像で作業認識を行う」と述べるだけでは、既存技術でどこまで可能なのかが不明確になりやすい。

そのため、まずSOTA手法や代表的な既存手法を整理し、実際に本研究の想定する条件に適用した場合に何ができて、何が難しいのかを確認したうえで、修論で扱うべき研究課題を定義したい。

## 研究全体の方向性

中心的な問いは以下のように置いている。

> 実環境の物流倉庫において、少ないアノテーションコストで作業認識を実現するにはどうすればよいか。

IMUベースでは、自己教師あり学習を用いて少量ラベル下での作業認識性能を評価してきた。
今後は、共同研究先で継続的に蓄積されている映像データも活用し、同じ「低アノテーションコスト」という観点から映像ベース作業認識へ拡張したい。

最終的に目指しているのは、倉庫内で日常的に蓄積される映像から、作業者ごとの作業状態を低コストに推定し、将来的には人の作業モデル化やシミュレーション連携に使えるデータへ変換することである。
ただし、そこへ直接進む前に、既存の人物追跡・行動認識技術でどこまで達成できるのかを確認し、既存技術の単純適用では残る課題を明確にする必要がある。

## なぜ今は調査をしているか

今回の調査の目的は、単なる関連研究の網羅ではなく、研究課題の妥当性を確認することである。

具体的には、以下を明らかにしたい。

- 俯瞰固定カメラ映像から複数人物を検出・追跡することは、既存手法でどこまで可能か。
- 各人物IDに対して行動ラベルや作業ラベルを付与することは、既存のaction recognition / action detection手法でどこまで可能か。
- 物流倉庫のような実環境に適用したとき、遮蔽、交差、類似作業服、画角外退出、長時間作業、作業境界の曖昧さがどの程度問題になるか。
- 少量アノテーションという条件下で、自己教師あり学習、弱教師あり学習、擬似ラベルなどがどこまで使えるか。
- 既存技術で十分解ける部分と、修論として新規性を出すべき部分を切り分ける。

特に、査読対応を見据えると、「既存手法を適用すればできるのではないか」という指摘に対して、本研究の条件ではどこに未解決性があるのかを説明できる必要がある。
そのため、SOTA手法を調査し、可能であれば一部はbaselineとして実際に適用することも検討している。

## 現在調査している領域

### 1. Multi-object Tracking / Multi-person Tracking

人物検出とID追跡の領域。
ByteTrack、BoT-SORT、OC-SORT、FairMOTなどが代表的。

わかってきたこと:

- 人物boxとID軌跡を得る技術はかなり成熟している。
- 一方で、これらの手法は基本的に作業ラベルを出さない。
- 倉庫のように作業服が似ている、遮蔽が多い、人物が交差する環境ではID維持が課題になりやすい。

### 2. Person Re-identification / Top-view ReID

人物が一度画角外に出たり、別カメラに移ったりした場合に同一人物を識別する領域。

わかってきたこと:

- 一般的なReIDは正面・側面の人物画像を前提にしたものが多い。
- 俯瞰視点では見た目の特徴が少なく、作業服が似ている環境では難しい可能性がある。
- 倉庫映像では、外観だけでなく軌跡や作業領域の情報も使う必要がありそう。

### 3. Video Action Recognition / Spatio-temporal Action Detection

映像中の人物行動を認識する領域。
SlowFast、VideoMAE、TubeR、EVAD、STARなどが関連する。

わかってきたこと:

- 短い映像クリップに対する行動認識はかなり進んでいる。
- 人物ごとのboxやtubeletに行動ラベルを付ける研究もある。
- ただし、物流倉庫のように数分から数十分続く作業を、同一人物IDに紐づけて認識する設定とはまだ距離がある。

### 4. Trajectory-centric Action Recognition

追跡で得た人物軌跡やtrackletを単位として行動認識する方向。
LARTなどが近い。

わかってきたこと:

- tracking結果をaction recognitionに使う発想は、本研究設定と相性が良い。
- 歩行、停止、移動、滞留のような粗い行動は軌跡から扱いやすい。
- 一方で、検品・仕分けのような作業は、軌跡だけではなく、棚・台車・作業台・商品などの文脈が必要になる可能性が高い。

## 特徴表現の観点での整理

現時点では、映像ベース作業認識に使う特徴をまだ決め打ちしていない。
関連研究を整理すると、主に以下の特徴表現が使われている。

| 特徴表現 | 関連する研究例 | 本研究への示唆 |
|---|---|---|
| RGB動画特徴 | SlowFast, VideoMAE, TubeR, EVAD, STAR | 一般的な行動認識・時空間行動検出では主流。ただし俯瞰倉庫映像では手元や作業物体が小さく見える可能性がある。 |
| 人物検出・外観特徴 | FairMOT, ByteTrack, BoT-SORT, OC-SORT | 複数人物のID追跡の基盤になる。ただし出力はbox + IDであり、作業ラベルは出ない。 |
| 軌跡・tracklet特徴 | LART, TrackTacular, MVTrajecter | 歩行、搬送、滞留、エリア移動などには有効そう。作業認識に接続しやすい。 |
| 骨格・pose特徴 | PoseC3D, LART, HA4M, CarDA | 身体動作の違いを見るには有効。ただし俯瞰・遮蔽・低解像度で推定が不安定になる可能性がある。 |
| object / zone context | MERL Shopping, RetailAction, IKEA ASM, Assembly101 | 検品・仕分けのような細かい作業意味には、棚・商品・台車・作業台などの文脈が必要になりそう。 |
| multimodal特徴 | HA4M, CarDA, IMU研究との接続 | IMUやRGB-Dを組み合わせることで低アノテーション化や弱教師化に使える可能性がある。 |

この整理から、まずは人物追跡で得られるtracklet / trajectoryを基盤にし、その上でRGB特徴、pose特徴、zone / object contextをどこまで足す必要があるかを検討する方針が自然だと考えている。

### 5. Industrial / Retail HAR

工場・小売・組立作業など、実環境に近い行動認識の研究。
MERL Shopping、RetailAction、IKEA ASM、Assembly101、CarDAなどを確認している。

わかってきたこと:

- 実作業に近いラベルを扱う研究は存在する。
- ただし、作業台中心、少人数、組立作業、店舗行動などが多く、物流倉庫の多人数俯瞰映像とは完全には一致しない。
- 公開データセットだけで本研究設定を完全に満たすものは少なそう。

## 現時点での整理

既存技術でできそうなこと:

- 人物検出
- 単一カメラまたは複数カメラでの人物追跡
- 歩行・停止・移動・滞留などの粗い行動認識
- 短い映像クリップの行動認識
- 自己教師あり学習による動画表現学習

まだ課題が残りそうなこと:

- 俯瞰固定カメラでの細かい作業認識
- 複数人物の長時間ID維持
- 似た作業服・遮蔽・交差への対応
- 数分から数十分続く作業区間の認識
- フレーム単位の密なアノテーションを避けた作業認識
- 軌跡、姿勢、作業領域、物体文脈をどう組み合わせるか

## 注目している研究方向

現時点では、新しいtrackerを一から作るより、既存の人物追跡技術で得られる軌跡やtrackletを利用して作業認識を行う方向に注目している。

仮の方向性:

> 俯瞰固定カメラ映像から得られる人物軌跡またはtrackletを用いて、少量の作業区間ラベルから検品・仕分け・搬送を認識する。

この方向であれば、IMU研究で扱ってきた「少量ラベル下での作業認識」と一貫性があり、映像データが継続的に蓄積される現場のニーズにも合う。

## 今後の予定

1. 主要論文を5から10本程度に絞って本文を確認する。
2. SOTAまたは代表的な既存手法を、倉庫映像に近い条件でbaselineとして適用できるか検討する。
3. 既存技術で実現できる範囲と、研究課題として残る範囲を明確にする。
4. 自前の倉庫映像で評価可能な最小タスクを定義する。
5. IMUベースの研究と、映像ベースの研究を「低アノテーションコストな作業認識」という軸で接続する。

## 報告時の一言まとめ

現在は、映像ベース作業認識を修論に入れるために、まず既存技術でどこまで実現できるかを調査している。
以前の査読対応で手法の新規性や既存研究との差分を問われた経験から、今回はSOTA手法や代表的な既存手法を整理し、それらを物流倉庫の俯瞰映像に適用した場合に本当に解けるのか、どこに課題が残るのかを確認したい。
現時点では、人物追跡や短時間の行動認識はかなり進んでいる一方、長時間IDを維持しながら少量ラベルで作業区間を認識する部分には、まだ研究課題が残りそうだと整理している。
