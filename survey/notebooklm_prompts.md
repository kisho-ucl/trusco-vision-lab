# NotebookLM Prompts

## Quick Screening Prompt

多くの論文をざっと見て、読む価値があるものを選ぶための短めプロンプト。

```text
私の研究テーマは、物流倉庫の俯瞰固定カメラ映像から複数人物を追跡し、各人物IDに対して検品・仕分け・搬送などの作業ラベルを少ないアノテーションで推定することです。

この論文を、私の研究に関係するかどうかを判断するために短く整理してください。

以下の形式で出力してください。

1. 何の論文か
   - タスク:
   - 入力:
   - 出力:
   - 主な特徴表現: RGB / pose / trajectory / tracklet / object / zone / multimodal のどれか

2. 私の研究に近い点
   - 俯瞰映像:
   - 複数人物:
   - ID追跡:
   - 行動・作業認識:
   - 少量ラベル:
   - 実環境・産業応用:

3. 私の研究とズレる点

4. この論文から分かる「既存技術でできること」

5. この論文から見える「まだ難しいこと」

6. 判断
   - 必読 / 精読候補 / 参考程度 / 関係薄い
   - 理由:
```

## Multi-paper Screening Prompt

複数PDFを入れて、次に読むべきものを選ぶための短めプロンプト。

```text
私の研究テーマは、物流倉庫の俯瞰固定カメラ映像から複数人物を追跡し、各人物IDに対して検品・仕分け・搬送などの作業ラベルを少ないアノテーションで推定することです。

アップロードした論文を横断して、私の研究に関係する順に整理してください。

表形式で出力してください。

列:
- 論文名
- 研究領域
- 入力
- 出力
- 主な特徴表現: RGB / pose / trajectory / tracklet / object / zone / multimodal
- 俯瞰・複数人物・ID追跡との関係
- 作業認識との関係
- 少量ラベルとの関係
- 私の研究との近さ: 高 / 中 / 低
- 次の判断: 必読 / 精読候補 / 参考程度 / 関係薄い

最後に、次にちゃんと読むべき論文を5本以内で選び、理由を短く書いてください。
```
